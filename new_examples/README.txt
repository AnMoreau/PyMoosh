The scripts in this folder are designed as simple use cases that can be easily copy/pasted and modified.
They are very similar to the Jupyter Notebooks provided with the code, so have a look there if you need more details.

RUNNING CODE:

(local installation)
If not using the latest PyPi version (typically, because you want access to functions that are on github but not yet PyPi):
Just copy the PyMoosh folder in your working directory, and run the scripts there!
